命名

python模块解释器默认从当前文件夹开始寻找是否有在模块中调用的库，如果你的package命名和内置的库名称一致，就会优先调用用户定义的库，这个时候就会出错

定义根目录

自带的根目录是src，如果你在下面还建立了多个package，在引入第三方的库时会有错误提示但是运行不会出问题，使用pycharm在当前的package下mark directory as source即可
