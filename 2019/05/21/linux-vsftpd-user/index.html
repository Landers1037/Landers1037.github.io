用户配置

user_config_dir是用户配置信息目录
cd /etc/vsftpd mkdir userconfig
进入userconfig下创建配置文件
nano name

systemctl restart vsftpd.service

注意

chroot_local_user=YES一旦添加，用户根目录就会被锁定为/home/name不能改变
如果想要新建的用户根目录为pub公共目录下，必须注释上述项

更改根目录所属

cd /var/ftp chown -R name:name name
